# Job Tracker Pro

A professional desktop app built with Python that helps users track job applications.

## Features

* Add/Edit/Delete job applications
* Search companies
* Dark Mode
* Company website links
* CSV Export
* Windows installer

## Built With

* Python
* Tkinter
* PyInstaller
* Inno Setup
